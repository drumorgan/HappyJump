Sign in to Faction Events

Faction Events keeps your Torn API key encrypted on our server so you don't have to re-enter it every visit. Sign in once and your counts refresh automatically when anyone loads the event page.

No key? Click here to create a Custom Key — it only needs your name, level, faction, log, and calendar. Your key is encrypted server-side and tied only to Faction Events; signing out deletes it.

Create a New Event

Set the drug, the start date and how long the event runs. You'll get a shareable link to send to your faction.

Find item IDs on the Torn API explorer (selection: items) or browse Torn Stats. The counter only matches "use" log entries — make sure the item is something players actually consume.

Pick when YOUR count starts inside the event window. Defaults to the 15-minute slot at-or-before now (so right when you create the event, the latest pre-event slot is selected). 15-minute slots from event start to event end, in your local time.
